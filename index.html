Aurveil

Visual wellness awareness in a private, simple flow.

Aurveil helps people observe visible changes in skin, eyes, posture, and overall presentation. It is not a diagnosis tool and does not replace licensed medical care.

Features for patients, doctors, and clinics

Designed for practical visual check-ins and responsible communication.

Guided scan flow

Step-by-step prompts for alignment, posture, and stillness.

Privacy-first by design

Visible privacy badge and no live video storage in normal use.

Non-diagnostic language

Output stays in visual observation and wellness awareness.

Local scan history

Timeline of prior scans stored in local browser storage.

Before vs after

Side-by-side visual comparison to track visible changes.

Profile and notes

Local-only user profile plus doctor notes per selected scan.

How it works

A short flow from camera setup to report review.

1. Open scanner

Start scan and allow camera access.

2. Align face

Use the frame guide and real-time feedback.

3. Capture frame

Capture a still image for visual wellness analysis.

4. Review report

Read observations, add notes, and compare scan history.
